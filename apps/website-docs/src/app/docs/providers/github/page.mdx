# Github

Github OAuth Provider allows users to log in to third-party applications using their Github account. It follows the OAuth 2.0 standard, ensuring secure access to Github services without sharing passwords.

### How to use?

Ether Auth allows you to configure Github Provider. You must set clientId, clientSecret and callbackUrl.

<Code>
  <ExpressCode type="ExpressCode">
  
    ```ts filename="auth.ts"
    import { expressAuth, githubProvider } from "@ether-auth/express";

    export const auth = expressAuth({
        providers: [githubProvider({
            clientId: process.env.GITHUB_CLIENT_ID,
            clientSecret: process.env.GITHUB_CLIENT_SECRET,
            callbackUrl: process.env.GITHUB_CALLBACK_URL
        })],
        adapter: //...
    })
    ```

  </ExpressCode>
